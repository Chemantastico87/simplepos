Preguntas Frecuentes

¿Tienes alguna pregunta sobre el funcionamiento de SimplePos? Aquí tienes las respuestas.

¿Funciona sin conexión a internet?
Sí, SimplePos es una aplicación nativa instalable y funciona al 100% offline. Toda la base de datos de productos, inventario y ventas se almacena localmente en tu ordenador. Solo necesitas conexión si deseas configurar el envío automático de reportes de cierre de caja o las alertas de stock a tu email.
¿Realmente es un pago único? Sin cuotas mensuales?
Sí. Una vez compras la licencia estándar de SimplePos, el software es tuyo para siempre. No cobramos comisiones por tus transacciones, ni cuotas mensuales de mantenimiento, ni renovaciones de licencia obligatorias.
¿Qué tipo de impresoras de tickets son compatibles?
SimplePos es compatible con la gran mayoría de impresoras térmicas de tickets (por ejemplo, de 58mm o 80mm) con conexión USB, red, o Bluetooth instaladas en el sistema operativo Windows mediante sus drivers estándar.
¿Puedo importar mi lista de productos actual?
Por supuesto. La base de datos es SQLite, y el software te permite exportar el inventario a formato CSV. También puedes solicitarnos el plan de soporte si deseas que nosotros mismos formateemos e importemos tu lista completa de artículos para empezar en minutos.
